# Grails3 REST API サンプル

##. Grails Install
 
https://grails.org/download.html から grails-3.1.6.zip をダウンロードして適当に展開、Path を通しておく。

##. 実行する

```
$ grails
grails> run-app
```
##. Swagger を開く

http://localhost:8080/api/index
